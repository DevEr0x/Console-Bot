Commands

Prefix: ?

Note: The bot uses this set prefix, there is currently no way to change it.

Using ?help [command] will give you more information.

Random - Info - Math - Moderation

Random Commands:

  • ?Ping - Replies with "Pong"
  • ?Hello - Replies with a greeting
  • ?On - A check to see if the bot is online
  • ?RateMe - Will rate you on a scale from 1-100
  • ?CoinFlip - Flips a coin
  • ?Joke - Tells a joke
  • ?8Ball - Ask it a question
  • ?Roll - Rolls a 6 sided die
  • ?Buttfuck - This is a weird one
  • ?Quote - Says a quote by a person you don't know
  • ?Avatar - Mention a user to display their avatar
  • ?KMN - Displays the suicide prevention hotline
  • ?Insult - Insults you
  • ?Night - Displays a goodnight message
  • ?Talk - Makes Console Bot send whatever you type.
  • ?Talkd - Does the same thing as the Say command, but it deletes your command.
  • ?SmashPass - Replies with a response to the question: Smash or Pass?
  • ?NSFW - Naughty Naughty
  • ?Potato - For a friend...
  • ?Pun - Who doesn't love puns? (Everyone)
  • ?Joy - I hope this brings you some "Joy".
  • ?Marco - Polo!
  • ?LovesMe - Ever played "He/She loves me, he/she loves me not"? It's that.
  • ?Kms - Poetic (Literally replies with some poem type thing)
  • ?SendHelp - I literally can't even.

Info Commands:

  • ?Info - Gives some info about the bot.
  • ?Server - Displays some Server Info
  • ?Help - Displays all the commands
  • ?Invite - Gives an invite to the support server
  • ?Website - Sends you a link to this page.
  • ?Stalk - Gets you more information about a specific user.

Math Commands:

  • ?Add - Adds 2 numbers together
  • ?Sub - Subtracts the second number from the first
  • ?Remainder - Divides two numbers and displays the remainder.
  • ?Multiply - Multiplies 2 numbers together
  • ?Divide - Divides the first number by the second number

Moderation Commands:

All of these commands require "Admin" role to work

  • ?Kick - Kicks a member.
  • ?Ban - Bans a member.
  • ?Prune - Deletes the specified amount of messages [1-500]
  • ?Magicnick - Gives the user a magic role which will change color every time a user with the role sends a message.
